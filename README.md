# DeviceDrivers_Practice
This repo is to save the practiced device drivers codes

## Basic Commands
* modinfo - to get the metadata of kernel module
	`$ modinfo ./hello.ko`
* insmod - to insert the kernel module
	`$sudo insmod ./hello.ko`
* lsmod - to check the list of kernel modules
	`$sudo lsmod`
* rmmod - to remove the kernel module
	`$sudo rmmod hello`
